Introduction

Blogs are probably a good way to show case the bioinformatic learning journey. A free blog can be created using github and Hugo themes. Following instructions were provided for a Mac.

GitHub Setup

If not already, create a github account. Make two repositories in your account.

  • 1 Your github account name followed by github.io. This folder visibility is set to public.
  • 2 Generic blog repository. Used to store all the code needed. This could be private.

Clone these repositories to your local computer

Hugo Setup

Install Hugo using Home brew. Here are the instruction on how to install home brew on your mac.

brew install hugo

Once installed. Get the boiler plate code required into the clone blog folder.

hugo new site blog --force

Find a theme that fits your blog from HUGO themes. Here hugo-blog-awesome is being used.

# change dir to the blog folder
cd blog

# get the theme to the themes folder
git clone https://github.com/hugo-sid/hugo-blog-awesome.git themes/hugo-blog-awesome

Access the blog folder using VScode and edit the hugo.toml or config.toml

# Access vscode In the blog folder 
code .
  • Open hugo.toml
  • Change base URL to the github account name followed by github.io
  • Add title for your blog
  • Add the theme –> theme = ‘hugo-blog-awesome’

First Page

Make a blog post using hugo

hugo new content posts/blog_setup.md

This creates a markdown which can be accessed using the VScode and content is added. Same rules of markdown apply.

Check how the post looks

hugo server -D

Access the page on the local computer using http://localhost:1313

Posting

If everything looks good the post needs to be published.

  • Remove draft= true from the page.
  • Connect the blog folder to the github account name followed by github.io
# change directory to kra277.github.io

# add main branch
git checkout -b main

# add a README.md
touch README.md

# Commit and Push to Github
git add .
git commit -m "adding readme.md"
git push origin main
  • Get back to the blog directory and publish the post
hugo -t hugo-blog-awesome

Once the post is published. Go into the public directory. Commit and Push to Github

git add .
git commit -m "firt post commit"
git push origin main

This will push the static site to the main branch of the blogpost. This may take a few minutes after which the site would be live with the contents created.
